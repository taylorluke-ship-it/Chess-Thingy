🧟 ZOMBIE ARCHITECTURE - FAULT TOLERANCE
─────────────────────────────────────────────────────────────────────────────

No subsystem failure crashes the server. Every handler wrapped in try/except:

  Stockfish fails?
    → Automatic fallback to FallbackAI ✅

  Database unavailable?
    → In-memory cache, retry later ✅

  Room corrupts?
    → Isolated to that room, others continue ✅

  WebSocket error?
    → Malformed packets rejected safely ✅

  Player disconnects?
    → Graceful cleanup, game preserved ✅

  Move validation fails?
    → Error sent to client, server continues ✅

  AI move fails?
    → Game continues with fallback ✅

  Chat persistence fails?
    → Message delivered, logged, retry later ✅

Result: Server stays online even when individual systems fail
